Creating "Real" Buttons

Usually the best way to put a button in an interface is to use a regular button, using either the <input> tag or the <button> tag. Most browsers give a three-dimensional look to their buttons. You can override the browser's native styles with CSS if you like, as we have done in these examples.

Submit button:

<input type="submit" name="button" id="submitbutton" value="Submit">

Image button:

<input type="image" src="search.png" alt="search" name="searchbutton" id="searchbutton">

Button with text:

<button>Expand all</button>

Buttons with images:

<button><img src="icon-bold.png" alt="bold" width="16" height="16"></button>
<button><img src="icon-italic.png" alt="italic" width="16" height="16"></button>

Button with an image and text:

<button>Preferences <img src="icon-prefs.png" alt="" width="16" height="16"></button>
