  • Functionality
    • Provide a tabbed UI, where the user can switch between tabs to view different sets of content
    • There are two parts to the UI:
      • The tab list, which shows the tab
      • The tab panels, which show the content associated with the tabs
    • Within a tab panel, UI is normal for the page; the “special” UI is within the tab list
    • Keyboard
      • Tab list presents only one tab stop, for the active tab
      • You can move between tabs with left/right, up/down, or ctrl+pgup/ctrl+pgdn
        • Moving between tabs changes the active tab immediately
        • Reaching the end wraps around to the other end
    • Mouse
      • Clicking on a tab activates the clicked-on tab
  • Implementation
    • HTML
      • Tab list is a ul, with listitems for each tab
        • ul has role=tablist
        • list items
          • have role=tab
          • have aria-controls= the id of the tabpanel tied to the tab
          • for the active tab
            • aria-selected=true
            • tabindex=0
          • for inactive tabs
            • aria-selected=false
            • tabindex=-1
      • Tab panels are divs within a grouping div
        • No special markup for grouping div
        • Each tab panel
          • has role=tabpanel
          • active tab
            • has CSS class that makes it visible
            • aria-hidden=false
          • inactive tabs
            • Not visible via CSS
            • aria-hidden=true
    • Javascript
      • Initialize event handlers
        • On mouse click, focus clicked tab
        • On keydown
          • If left, up, ctrl-pageup, focus previous tab or last if no previous
          • If right, down, ctrl-pgdn, focus next tab or first if no next
      • Focus tab logic
        • Find the current active tab and inactivate it
          • For the tab in the tab list
            • set CSS classes appropriately
            • set aria-selected=false
            • set tabindex=-1
          • For the tabpanel
            • Set CSS appropriately
            • Set aria-hidden=true
        • Find the new tab and activate it
          • For the tab in the tab list
            • set CSS classes appropriately
            • set aria-selected=true
            • set tabindex=0
          • For the tabpanel
            • Set CSS appropriately
            • Set aria-hidden=false
        • Focus on the new tab header
